npx vercel link
Then open .vercel/project.json → copy orgId and projectId.

2. Get the token: Vercel → Account Settings → Tokens → Create → copy it.

3. Add 3 GitHub secrets — repo → Settings → Secrets and variables → Actions → New repository secret:

VERCEL_TOKEN = the token
VERCEL_ORG_ID = orgId
VERCEL_PROJECT_ID = projectId
4. Push everything to main:


git add .
git commit -m "Add BDD E2E and CI deploy gate"
git push
5. Watch it: GitHub repo → Actions tab → click the running job. test must pass before deploy runs.
